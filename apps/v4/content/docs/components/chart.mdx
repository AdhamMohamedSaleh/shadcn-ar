تعرّف على **المخططات**. مجموعة من مكونات المخططات التي يمكنك نسخها ولصقها بتطبيقاتك.

تم تصميم المخططات لتبدو رائعة مباشرة دون تعديل. وهي تعمل بشكل جيد مع بقية المكونات ويمكن تخصيصها بالكامل لتناسب مشروعك.

[تصفح مكتبة المخططات](/charts).

## المكون

نستخدم [Recharts](https://recharts.org/) تحت الغطاء.

صممنا مكون `chart` مع وضع التركيب في الاعتبار. **أنت تبني مخططاتك باستخدام مكونات Recharts وتستحضر المكونات المخصصة، مثل `ChartTooltip`، عندما تحتاج إليها فحسب**.

```tsx showLineNumbers /ChartContainer/ /ChartTooltipContent/
import { Bar, BarChart } from "recharts"

import { ChartContainer, ChartTooltipContent } from "@/components/ui/charts"

export function MyChart() {
  return (
    <ChartContainer>
      <BarChart data={data}>
        <Bar dataKey="value" />
        <ChartTooltip content={<ChartTooltipContent />} />
      </BarChart>
    </ChartContainer>
  )
}
```

نحن لا نلف Recharts، أي لن يتم حصرُك في تجريدات إضافية. عند صدور إصدار جديد من Recharts، يمكنك اتباع مسار الترقية الرسمي لمخططاتك.

**المكونات ملكك بالكامل**.

## أول مخطط

لننشئ مخططك الأول: سنقوم ببناء مخطط أعمدة، ونضيف شبكة، ومحور، وتلميح (tooltip)، ووسيلة إيضاح (legend).

<Steps>

<Step>ابدأ بتعريف بياناتك</Step>

البيانات التالية تمثل عدد مستخدمي سطح المكتب والجوال لكل شهر.

<Callout className="mt-4">**ملاحظة:** يمكن أن تكون بياناتك بأي شكل، استخدم خاصية `dataKey` لربط البيانات بالمخطط.</Callout>

  { month: "مارس", desktop: 237, mobile: 120 },

<Step>عرّف إعدادات المخطط Chart Config</Step>

## إعداد المخطط Chart Config

إعداد المخطط هو المكان الذي تحدد فيه التسميات والأيقونات والألوان للمخطط.

وقد تمّ فصله عن بيانات المخطط عن عمد.

يسمح لك هذا بمشاركة الإعدادات وألوان السمات بين المخططات. كما يعمل بشكل مستقل في الحالات التي توجد فيها بياناتك أو ألوان السمات عن بعد أو بتنسيق آخر.

```tsx showLineNumbers /ChartConfig/
import { Monitor } from "lucide-react"

import { type ChartConfig } from "@/components/ui/chart"

const chartConfig = {
  desktop: {
    label: "سطح المكتب",
    icon: Monitor,
    // لون مثل 'hsl(220, 98%, 61%)' أو 'var(--color-name)'
    color: "#2563eb",
    // أو كائن theme فيه مفتاحا 'light' و 'dark'
    theme: {
      light: "#2563eb",
      dark: "#dc2626",
    },
  },
} satisfies ChartConfig
```

يمكنك أيضًا تحديد ألوانك مباشرةً في إعدادات المخطط. استخدم صيغة اللون التي تفضلها.

```tsx showLineNumbers
const chartConfig = {

    color: "#2563eb",
  },

```

### استخدام الألوان

لاستخدام ألوان السمات في المخطط، استخدم الصيغة `var(--color-KEY)`.

#### المكونات

```tsx

```

#### بيانات المخطط

```tsx showLineNumbers
const chartData = [

  { browser: "safari", visitors: 200, fill: "var(--color-safari)" },

```

```tsx
<LabelList className="fill-[--color-desktop]" />
```

## التلميح Tooltip

يتكون تلميح المخطط من تسمية واسم ومؤشر وقيمة. يمكنك تخصيص تلميحك كما تريد باستخدام هذه العناصر.

<ComponentPreview
  name="chart-tooltip-demo"
  className="[&_.preview]:py-0"
  hideCode
/>

يمكنك تشغيل أو إيقاف أي منها عبر خصائص `hideLabel`, `hideIndicator`، وتخصيص المؤشر باستخدام خاصية `indicator`.

استخدم `labelKey` و `nameKey` لتخصيص مفتاح التسمية والاسم في التلميح.

المكونان `<ChartTooltip>` و `<ChartTooltipContent>` مخصصان للتلميحات المخصصة في مخططك.

```tsx
import { ChartTooltip, ChartTooltipContent } from "@/components/ui/chart"
```

```tsx
<ChartTooltip content={<ChartTooltipContent />} />
```

### الخصائص

استخدم هذه الخصائص لتخصيص التلميح:

| الخاصية        | النوع                    | الوصف                                    |
| :------------- | :----------------------- | :--------------------------------------- |
| `labelKey`     | string                   | مفتاح التسمية المستخدم.                  |
| `nameKey`      | string                   | مفتاح الاسم المستخدم.                    |
| `indicator`    | `dot` `line` أو `dashed` | نمط المؤشر في التلميح.                   |
| `hideLabel`    | boolean                  | إخفاء التسمية أم لا.                     |
| `hideIndicator`| boolean                  | إخفاء المؤشر أم لا.                      |

### الألوان

الألوان تتم قراءتها تلقائيًا من إعداد المخطط Chart Config.

### تخصيص

لاستخدام مفتاح مخصص للتسمية أو الاسم، استخدم خاصيتي `labelKey` و `nameKey`.

```tsx showLineNumbers /browser/
const chartData = [
  { browser: "chrome", visitors: 187, fill: "var(--color-chrome)" },
  { browser: "safari", visitors: 200, fill: "var(--color-safari)" },
]

const chartConfig = {
  visitors: {
    label: "إجمالي الزوار",
  },
  chrome: {
    label: "Chrome",
    color: "hsl(var(--chart-1))",
  },
  safari: {
    label: "Safari",
    color: "hsl(var(--chart-2))",
  },
} satisfies ChartConfig
```

```tsx
<ChartTooltip
  content={<ChartTooltipContent labelKey="visitors" nameKey="browser" />}
/>
```

سيظهر "إجمالي الزوار" كتسمية، و"Chrome"، و"Safari" كأسماء في التلميح.

```tsx
<ChartLegend content={<ChartLegendContent />} />

    label: "Chrome",
    color: "hsl(var(--chart-1))",
  },
  safari: {
    label: "Safari",
    color: "hsl(var(--chart-2))",
  },
} satisfies ChartConfig
```

```tsx
<ChartLegend content={<ChartLegendContent nameKey="browser" />} />
```

سيظهر "Chrome" و"Safari" كأسماء في وسيلة الإيضاح.
