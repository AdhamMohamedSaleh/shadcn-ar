### رابط

يمكنك استخدام خاصية `asChild` لجعل مكون آخر يبدو مثل شارة. إليك مثال على رابط يبدو مثل شارة.

```tsx showLineNumbers
import Link from "next/link"

import { Badge } from "@/components/ui/badge"

export function LinkAsBadge() {
  return (
    <Badge asChild>
      <Link href="/">شارة</Link>
    </Badge>
  )
}
```
