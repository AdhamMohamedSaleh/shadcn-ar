### التباعد

لتعيين التباعد بين العناصر، نستخدم فئة `pl-[VALUE]` المساعدة على `<CarouselItem />` و `-ml-[VALUE]` سالبة على `<CarouselContent />`.

<Callout className="mt-6">
  **لماذا:** حاولت استخدام خاصية `gap` أو تخطيط `grid` على `
  <CarouselContent />` لكنه تطلب الكثير من الحساب والجهد الذهني للحصول على التباعد الصحيح. وجدت أن فئات `pl-[VALUE]` و `-ml-[VALUE]` المساعدة أسهل بكثير في الاستخدام.

يمكنك دائمًا ضبط هذا في مشروعك الخاص إذا احتجت إلى ذلك.

</Callout>

## API

استخدم حالة وخاصية `setApi` للحصول على مثيل لواجهة برمجة تطبيقات الدائري.

<ComponentPreview
  name="carousel-api"
  title="دائري"
  description="دائري مع عداد شرائح."
/>

```tsx showLineNumbers {1,4,22}
import { type CarouselApi } from "@/components/ui/carousel"

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()
  const [current, setCurrent] = React.useState(0)
  const [count, setCount] = React.useState(0)

  React.useEffect(() => {
    if (!api) {
      return
    }

    setCount(api.scrollSnapList().length)
    setCurrent(api.selectedScrollSnap() + 1)

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1)
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

## الأحداث

يمكنك الاستماع إلى الأحداث باستخدام مثيل api من `setApi`.

```tsx showLineNumbers {1,4-14,16}
import { type CarouselApi } from "@/components/ui/carousel"

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()

  React.useEffect(() => {
    if (!api) {
      return
    }

    api.on("select", () => {
      // افعل شيئًا عند التحديد.
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

راجع [وثائق Embla Carousel](https://www.embla-carousel.com/api/events/) لمزيد من المعلومات حول استخدام الأحداث.

## الإضافات

يمكنك استخدام خاصية `plugins` لإضافة إضافات إلى الدائري.

```ts showLineNumbers {1,6-10}
import Autoplay from "embla-carousel-autoplay"

export function Example() {
  return (
    <Carousel
      plugins={[
        Autoplay({
          delay: 2000,
        }),
      ]}
    >
      // ...
    </Carousel>
  )
}
```
