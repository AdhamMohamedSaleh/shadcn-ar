## البنية

يتكون مكون `Sidebar` من الأجزاء التالية:

- `SidebarProvider` - Handles collapsible state.
- `Sidebar` - حاوية الشريط الجانبي.
- `SidebarHeader` و`SidebarFooter` - ثابتان أعلى وأسفل الشريط الجانبي.
- `SidebarContent` - محتوى قابل للتمرير.
- `SidebarGroup` - قسم داخل `SidebarContent`.
- `SidebarTrigger` - مُشغِّل الشريط الجانبي.

## المكونات

المكونات في `sidebar.tsx` مبنية لتكون قابلة للتكوين، أي أنك تبني الشريط الجانبي عن طريق تركيب المكونات المتوفرة معًا. كما أنها تتكامل جيدًا مع مكونات shadcn/ui الأخرى مثل `DropdownMenu` و`Collapsible` و`Dialog` وغيرها.

**إذا احتجت لتعديل الشفرة في `sidebar.tsx` فنشجعك على ذلك. الشفرة لك. استخدم `sidebar.tsx` كنقطة انطلاق وابنِ نسختك الخاصة.**

في الأقسام التالية، سنستعرض كل مكون وكيفية استخدامه.

## SidebarProvider

يُستخدم مكون `SidebarProvider` لتزويد مكون `Sidebar` بالسياق. يجب دائمًا إحاطة تطبيقك بمكون `SidebarProvider`.

### الخصائص

| الاسم           | النوع                      | الوصف                                         |
| -------------- | ------------------------- | --------------------------------------------- |
| `defaultOpen`  | `boolean`                 | الحالة الافتراضية لفتح الشريط الجانبي.       |
| `open`         | `boolean`                 | حالة فتح الشريط الجانبي (محكومة).            |
| `onOpenChange` | `(open: boolean) => void` | يضبط حالة فتح الشريط الجانبي (محكومة).       |

### العرض

إذا كان لديك شريط جانبي واحد في تطبيقك، يمكنك استخدام المتغيرين `SIDEBAR_WIDTH` و`SIDEBAR_WIDTH_MOBILE` في `sidebar.tsx` لتحديد عرض الشريط الجانبي.

```tsx showLineNumbers title="components/ui/sidebar.tsx"
const SIDEBAR_WIDTH = "16rem"
const SIDEBAR_WIDTH_MOBILE = "18rem"
```

لوجود عدة أشرطة جانبية في تطبيقك، يمكنك استخدام خاصية `style` لتحديد عرض الشريط الجانبي.

To set the width of the sidebar, you can use the `--sidebar-width` and `--sidebar-width-mobile` CSS variables in the `style` prop.

```tsx showLineNumbers title="components/ui/sidebar.tsx"
<SidebarProvider
  style={{
    "--sidebar-width": "20rem",
    "--sidebar-width-mobile": "20rem",
  }}
>
  <Sidebar />
</SidebarProvider>
```

سيؤدي هذا إلى التحكم في عرض الشريط الجانبي وكذلك المسافات في التخطيط.

### اختصار لوحة المفاتيح

يُستخدم المتغير `SIDEBAR_KEYBOARD_SHORTCUT` لتعيين اختصار لوحة المفاتيح لفتح الشريط الجانبي وإغلاقه.

لتحفيز الشريط الجانبي، استخدم الاختصار `cmd+b` على الماك و`ctrl+b` على ويندوز.

يمكنك تغيير اختصار لوحة المفاتيح عبر تحديث المتغير `SIDEBAR_KEYBOARD_SHORTCUT`.

```tsx showLineNumbers title="components/ui/sidebar.tsx"
const SIDEBAR_KEYBOARD_SHORTCUT = "b"
```

### الحالة المُستدامة

يدعم `SidebarProvider` استدامة حالة الشريط الجانبي عبر إعادة تحميل الصفحات والتهيئة على الخادم. يستخدم ملفات تعريف الارتباط لتخزين الحالة الحالية للشريط الجانبي. عند تغير الحالة، يتم تعيين ملف تعريف ارتباط افتراضي باسم `sidebar_state` بقيمة حالة الفتح/الإغلاق. ثم تتم قراءته في التحميلات اللاحقة لاستعادة الحالة.

To persist sidebar state in Next.js, set up your `SidebarProvider` in `app/layout.tsx` like this:

```tsx showLineNumbers title="app/layout.tsx"
import { cookies } from "next/headers"

import { SidebarProvider, SidebarTrigger } from "@/components/ui/sidebar"
import { AppSidebar } from "@/components/app-sidebar"

export async function Layout({ children }: { children: React.ReactNode }) {
  const cookieStore = await cookies()
  const defaultOpen = cookieStore.get("sidebar_state")?.value === "true"

  return (
    <SidebarProvider defaultOpen={defaultOpen}>
      <AppSidebar />
      <main>
        <SidebarTrigger />
        {children}
      </main>
    </SidebarProvider>
  )
}
```

يمكنك تغيير اسم ملف تعريف الارتباط عبر تحديث المتغير `SIDEBAR_COOKIE_NAME` في `sidebar.tsx`.

```tsx showLineNumbers title="components/ui/sidebar.tsx"
const SIDEBAR_COOKIE_NAME = "sidebar_state"
```

## Sidebar

المكون الرئيسي `Sidebar` المستخدم لعرض شريط جانبي قابل للطي.

```tsx showLineNumbers
import { Sidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  return <Sidebar />
}
```

### الخصائص

| الخاصية       | النوع                             | الوصف                              |
| ------------- | --------------------------------- | ---------------------------------- |
| `side`        | `left` أو `right`                 | جانب ظهور الشريط الجانبي.         |
| `variant`     | `sidebar` أو `floating` أو `inset` | نوع الشريط الجانبي.               |
| `collapsible` | `offcanvas` أو `icon` أو `none`    | حالة قابلية الطي للشريط الجانبي.  |

### side

استخدم خاصية `side` لتغيير جانب الشريط الجانبي.

القيم المتاحة هي `left` و`right`.

```tsx showLineNumbers
import { Sidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  return <Sidebar side="left | right" />
}
```

### variant

استخدم خاصية `variant` لتغيير نوع الشريط الجانبي.

القيم المتاحة هي `sidebar` و`floating` و`inset`.

```tsx showLineNumbers
import { Sidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  return <Sidebar variant="sidebar | floating | inset" />
}
```

<Callout>
  **ملاحظة:** إذا استخدمت نوع `inset` فتذكر إحاطة المحتوى الرئيسي
  بمكون `SidebarInset`.
</Callout>

```tsx showLineNumbers
<SidebarProvider>
  <Sidebar variant="inset" />
  <SidebarInset>
    <main>{children}</main>
  </SidebarInset>
</SidebarProvider>
```

### collapsible

استخدم خاصية `collapsible` لجعل الشريط الجانبي قابلًا للطي.

القيم المتاحة هي `offcanvas` و`icon` و`none`.

```tsx showLineNumbers
import { Sidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  return <Sidebar collapsible="offcanvas | icon | none" />
}
```

| الخاصية     | الوصف                                                        |
| ----------- | ------------------------------------------------------------ |
| `offcanvas` | شريط جانبي قابل للطي ينزلق من اليسار أو اليمين.              |
| `icon`      | شريط جانبي ينهار إلى أيقونات.                               |
| `none`      | شريط جانبي غير قابل للطي.                                    |

## useSidebar

تُستخدم دالة `useSidebar` للتحكم في الشريط الجانبي.

```tsx showLineNumbers
import { useSidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  const {
    state,
    open,
    setOpen,
    openMobile,
    setOpenMobile,
    isMobile,
    toggleSidebar,
  } = useSidebar()
}
```

| الخاصية         | النوع                      | الوصف                                          |
| --------------- | ------------------------- | ---------------------------------------------- |
| `state`         | `expanded` أو `collapsed` | الحالة الحالية للشريط الجانبي.                 |
| `open`          | `boolean`                 | ما إذا كان الشريط الجانبي مفتوحًا.             |
| `setOpen`       | `(open: boolean) => void` | يضبط حالة فتح الشريط الجانبي.                  |
| `openMobile`    | `boolean`                 | ما إذا كان الشريط الجانبي مفتوحًا على الجوال.  |
| `setOpenMobile` | `(open: boolean) => void` | يضبط حالة فتح الشريط الجانبي على الجوال.       |
| `isMobile`      | `boolean`                 | ما إذا كان العرض على الجوال.                   |
| `toggleSidebar` | `() => void`              | يبدّل حالة الشريط الجانبي على الحاسوب والجوال.  |

## التنسيق

إليك بعض النصائح لتنسيق الشريط الجانبي بناءً على حالات مختلفة.

- **تنسيق عنصر بناءً على حالة قابلية طي الشريط الجانبي.** التالي سيخفي `SidebarGroup` عندما يكون الشريط الجانبي في وضع `icon`.

```tsx
<Sidebar collapsible="icon">
  <SidebarContent>
    <SidebarGroup className="group-data-[collapsible=icon]:hidden" />
  </SidebarContent>
</Sidebar>
```

- **تنسيق إجراء القائمة بناءً على حالة زر القائمة النشطة.** التالي سيجبر إجراء القائمة على الظهور عندما يكون زر القائمة نشطًا.

```tsx
<SidebarMenuItem>
  <SidebarMenuButton />
  <SidebarMenuAction className="peer-data-[active=true]/menu-button:opacity-100" />
</SidebarMenuItem>
```

يمكنك العثور على مزيد من النصائح حول استخدام الحالات للتنسيق في [خيط على منصة X](https://x.com/shadcn/status/1842329158879420864).
