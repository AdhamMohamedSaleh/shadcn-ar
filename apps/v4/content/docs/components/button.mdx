## المؤشر

Tailwind v4 [تحول](https://tailwindcss.com/docs/upgrade-guide#buttons-use-the-default-cursor) من `cursor: pointer` إلى `cursor: default` لمكون الزر.

إذا كنت ترغب في الحفاظ على سلوك `cursor: pointer`، أضف الكود التالي إلى ملف CSS الخاص بك:

```css showLineNumbers title="globals.css"
@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}
```

### رابط

يمكنك استخدام خاصية `asChild` لجعل مكون آخر يبدو مثل زر. إليك مثال على رابط يبدو مثل زر.

```tsx showLineNumbers
import Link from "next/link"

import { Button } from "@/components/ui/button"

export function LinkAsButton() {
  return (
    <Button asChild>
      <Link href="/login">تسجيل الدخول</Link>
    </Button>
  )
}
```

## مرجع API

### زر

مكون `Button` هو غلاف حول عنصر `button` يضيف مجموعة متنوعة من الأنماط والوظائف.

| الخاصية   | النوع                                                                          | الافتراضي   |
| --------- | ----------------------------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "outline" \| "ghost" \| "destructive" \| "secondary" \| "link"` | `"default"` |
| `size`    | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`               | `"default"` |
| `asChild` | `boolean`                                                                     | `false`     |

## سجل التغييرات

### 2025-09-24 أحجام جديدة

لقد أضفنا حجمين جديدين لمكون الزر: `icon-sm` و `icon-lg`. تُستخدم هذه الأحجام لإنشاء أزرار أيقونات. لإضافتها، قم بتحرير `button.tsx` وأضف الكود التالي تحت `size` في `buttonVariants`:

```tsx showLineNumbers title="components/ui/button.tsx"
const buttonVariants = cva("...", {
  variants: {
    size: {
      // ...
      "icon-sm": "size-8",
      "icon-lg": "size-10",
      // ...
    },
  },
})
```
