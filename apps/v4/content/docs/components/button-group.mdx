## الاستخدام

```tsx
import {
  ButtonGroup,
  ButtonGroupSeparator,
  ButtonGroupText,
} from "@/components/ui/button-group"
```

```tsx
<ButtonGroup>
  <Button>الزر 1</Button>
  <Button>الزر 2</Button>
</ButtonGroup>
```

## إمكانية الوصول

- يحتوي مكون `ButtonGroup` على السمة `role` المعينة إلى `group`.
- استخدم <Kbd>Tab</Kbd> للتنقل بين الأزرار في المجموعة.
- استخدم `aria-label` أو `aria-labelledby` لتسمية مجموعة الأزرار.

```tsx showLineNumbers
<ButtonGroup aria-label="مجموعة الأزرار">
  <Button>الزر 1</Button>
  <Button>الزر 2</Button>
</ButtonGroup>
```

## ButtonGroup مقابل ToggleGroup

- استخدم مكون `ButtonGroup` عندما تريد تجميع الأزرار التي تؤدي إجراءً.
- استخدم مكون `ToggleGroup` عندما تريد تجميع الأزرار التي تبدل حالة.

## مرجع API

### ButtonGroup

مكون `ButtonGroup` هو حاوية تجمع الأزرار ذات الصلة معًا بتصميم متسق.

| الخاصية | النوع | الافتراضي |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

```tsx
<ButtonGroup>
  <Button>الزر 1</Button>
  <Button>الزر 2</Button>
</ButtonGroup>
```

قم بتداخل مجموعات أزرار متعددة لإنشاء تخطيطات معقدة مع تباعد. انظر مثال [المتداخل](#nested) لمزيد من التفاصيل.

```tsx
<ButtonGroup>
  <ButtonGroup />
  <ButtonGroup />
</ButtonGroup>
```

### ButtonGroupSeparator

يقوم مكون `ButtonGroupSeparator` بتقسيم الأزرار بصريًا داخل مجموعة.

| الخاصية | النوع | الافتراضي |
| ------------- | ---------------------------- | ------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` |

```tsx
<ButtonGroup>
  <Button>الزر 1</Button>
  <ButtonGroupSeparator />
  <Button>الزر 2</Button>
</ButtonGroup>
```

### ButtonGroupText

استخدم هذا المكون لعرض نص داخل مجموعة أزرار.

| الخاصية | النوع | الافتراضي |
| --------- | --------- | ------- |
| `asChild` | `boolean` | `false` |

```tsx
<ButtonGroup>
  <ButtonGroupText>نص</ButtonGroupText>
  <Button>زر</Button>
</ButtonGroup>
```

استخدم خاصية `asChild` لعرض مكون مخصص كنص، على سبيل المثال تسمية.

```tsx showLineNumbers
import { ButtonGroupText } from "@/components/ui/button-group"
import { Label } from "@/components/ui/label"

export function ButtonGroupTextDemo() {
  return (
    <ButtonGroup>
      <ButtonGroupText asChild>
        <Label htmlFor="name">نص</Label>
      </ButtonGroupText>
      <Input placeholder="اكتب شيئًا هنا..." id="name" />
    </ButtonGroup>
  )
}
```
